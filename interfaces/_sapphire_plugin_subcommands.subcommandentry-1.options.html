Interface Options<ArgType, CommandType>

The options for a SubCommandEntry.

property

input Input represents the subcommand that the user will type in.

property

output Output represents the method/command called for the subcommand.

example

    input: ({ message }) => message.resolveKey('subcommands:set'),
    output: 'set'
}]
